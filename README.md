# Nutrition PDF → Illustrator Automation

## 🚀 Features
- Extract nutrition data from PDF
- Convert to structured CSV
- Auto-import into Illustrator
- Automate label design

## 🛠 Tech Stack
- Python (Streamlit)
- pdfplumber
- Adobe Illustrator JSX

## ⚙️ Usage
1. Run Streamlit app
2. Upload PDF
3. Download CSV
4. Run Illustrator script

- Batch automation
